Logging#

Logging configuration and utilities.

forge.util.logging.get_logger(level: str | None = None) → Logger[source]#

Get a logger with a stream handler.

Parameters:

level (Optional[str]) – The logging level. See https://docs.python.org/3/library/logging.html#levels for list of levels.

Example

>>> logger = get_logger("INFO")
>>> logger.info("Hello world!")
INFO:torchtune.utils._logging:Hello world!
Returns:

The logger.

Return type:

logging.Logger

Parameters:

  • logger (logging.Logger) – The logger.

  • msg (str) – The warning message.

  • level (int) – The logging level. See https://docs.python.org/3/library/logging.html#levels for values. Defaults to logging.INFO.

forge.util.logging.log_once(logger: Logger, msg: str, level: int = 20) → None[source]#

Logs a message only once. LRU cache is used to ensure a specific message is logged only once, similar to how warn() works when the once rule is set via command-line or environment variable.

Parameters:

  • logger (logging.Logger) – The logger.

  • msg (str) – The warning message.

  • level (int) – The logging level. See https://docs.python.org/3/library/logging.html#levels for values. Defaults to logging.INFO.
